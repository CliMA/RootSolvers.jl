RootSolvers.jl

A simple GPU-capable root solver package.

Usage

using RootSolvers

sol = find_zero(x -> x^2 - 100^2,
                SecantMethod{Float64}(0.0, 1000.0),
                CompactSolution());
RootSolvers.CompactSolutionResults{Float64}(99.99999999994358, true)
